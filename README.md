# Beginner
一个原生js来实现的开始页,起始页,毫秒级响应,支持百度,必应搜索,还有雅兰/皙白模式切换,名言警句,同时又取色器功能,文件但只占34kb,还没进行压缩,等开发完成应该可以20kb以内
​
